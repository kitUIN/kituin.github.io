from PicImageSearch import TraceMoe,Network
from PicImageSearch.model import TraceMoeResponse
from loguru import logger

async with Network() as client:
    tracemoe = TraceMoe(client=client)
    resp = await tracemoe.me(key)
    logger.info(resp.id)# IP 地址（访客）或电子邮件地址（用户）
    logger.info(resp.priority) # 优先级
    logger.info(resp.concurrency) # 搜索请求数量
    logger.info(resp.quota) # 本月的搜索配额
    logger.info(resp.quotaUsed)# 本月已经使用的搜索配额

情境

以Demo 中的代码为例子

变量 内容 类型
resp.origin 原始数据 dict
resp.count 搜索结果数量 int
resp.frameCount 搜索的帧总数 int
resp.error 错误信息 str
resp.raw 结果返回值（具体见下表） list

提示

resp.raw 存储了所有的返回结果
例如 resp.raw[0] 内存放了第一条搜索结果

以下列表以resp.raw[0]为例

变量 内容 类型
resp.raw[0].origin 原始数据 dict
resp.raw[0].anilistID 匹配的Anilist ID (opens new window) int
resp.raw[0].title 番剧名字(包含国际,罗马音,英文) ( anilistInfo (opens new window) 设为 True (默认)时) dict
resp.raw[0].title_native 番剧国际名字 ( anilistInfo (opens new window) 设为 True (默认)时) str
resp.raw[0].title_romaji 番剧罗马音名字 ( anilistInfo (opens new window) 设为 True (默认)时) str
resp.raw[0].title_english 番剧英文名字 ( anilistInfo (opens new window) 设为 True (默认)时) str
resp.raw[0].title_chinese 番剧中文名字 ( anilistInfo (opens new window) 设为 True (默认)并且chineseTitle (opens new window) 设为 True (默认)时) str
resp.raw[0].idMal 匹配的MyAnimelist ID (opens new window) ( anilistInfo (opens new window) 设为 True (默认)时) int
resp.raw[0].synonyms 备用英文标题 ( anilistInfo (opens new window) 设为 True (默认)时) list
resp.raw[0].isAdult 是否R18 ( anilistInfo (opens new window) 设为 True (默认)时) bool
resp.raw[0].From 匹配场景的开始时间 int
resp.raw[0].To 匹配场景的结束时间 int
resp.raw[0].filename 找到匹配项的文件名 str
resp.raw[0].episode 估计的匹配的番剧的集数 int
resp.raw[0].similarity 相似度，相似性低于 87% 的搜索结果可能是不正确的结果 float
resp.raw[0].image 预览图片地址 str
resp.raw[0].video 预览视频地址 str
resp.raw[0].anime_info 标题信息 dict
